# Background Task Processor

A Spring Boot based background task processing system using Redis queues.

## Features
- Asynchronous task processing
- Redis-based task queue
- Task status tracking
- REST APIs for submitting and monitoring tasks

## Tech Stack
- Java
- Spring Boot
- Redis
- Maven

## How it Works
Tasks are pushed into a Redis queue and processed asynchronously by worker services.

## Example Tasks
- SUM
- MULTIPLY
